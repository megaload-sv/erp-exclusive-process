# PR-007 — Universal Component Composition Engine

## Objective

Introduce a universal tree model so TraceOps components can be composed, navigated and inspected as structured application interfaces instead of isolated UI elements.

## Capability acquired

```php
$page = new PageComponent();
$toolbar = new ToolbarComponent();
$button = new ButtonComponent(['label' => 'Guardar']);

$page->addChild($toolbar);
$toolbar->addChild($button);

$button->parent();
$button->root();
$button->depth();
$button->path();
```

## Delivered

- `NodeInterface` as the universal composition contract
- `AbstractNode` with parent and child relationship management
- `NodeCollection` with deterministic identity-based membership
- Safe reparenting and node removal
- Circular tree protection
- Root, depth and path navigation
- `TreeWalker` with depth-first traversal
- Search by predicate, node name and node type
- `BaseComponent` integration with the universal node model
- Unit tests for composition, navigation, reparenting, traversal and cycle prevention

## Architectural flow

```text
NodeInterface
      ↓
AbstractNode
      ↓
BaseComponent
      ↓
Page / Card / Toolbar / Table / Button
      ↓
Universal Component Tree
      ↓
Renderer / Studio / AI / Inspector
```

## Acceptance criteria

- Components can contain child nodes.
- Every attached child knows its parent.
- Nodes expose their root, depth and path.
- Moving a node automatically detaches it from its previous parent.
- Removing a node clears its parent relationship.
- Duplicate child instances are not added twice.
- Circular composition is rejected.
- Trees can be traversed recursively in deterministic depth-first order.
- Nodes can be searched by predicate, name and concrete type.
- Existing component construction, normalization and rendering APIs remain compatible.

## Deferred

- Recursive rendering of child components
- Named slots such as header, body and footer
- Visitor interface and mutation pipeline
- Stable node identifiers
- Tree serialization to JSON
- Render middleware pipeline
