# Architecture Decision Records

Este directorio contiene las decisiones arquitectónicas relevantes de TraceOps ERP.

## Convención

Cada ADR debe incluir:

- contexto;
- decisión;
- consecuencias positivas y riesgos;
- alternativas consideradas;
- estado y fecha.

## Estados

- Propuesto
- Aceptado
- Reemplazado
- Rechazado
- Obsoleto
